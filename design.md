
# AI Prompt to Design Home Page for Hotel Managers Conference Africa

## Context

## Requirements

### Header
- Logo: Use `https://hotelmanagersconference.com/landingpage/images/hmc_logo.png` as the site logo
- Navigation menu items (in order):
  1. About
  2. Speakers
  3. Buy Tickets
  4. Awards
  5. Partner Hotels
  6. Gallery
- Style the header to match the existing landing page's design system:
  - Colors: Use the same maroon/gold/cream color palette as landing.html
  - Fonts: Use Playfair Display for headings, Inter for body text
  - Layout: Responsive, mobile-friendly

### Footer
- Reuse the **exact footer** from the existing landing.html
- Keep all footer content: Contact info, quick links, copyright, etc.

### Design System (from landing.html)
- **Colors**:
  - Maroon: #1a0510, #2d0a1e, #4a1030, #6b1a43

  - Ink: #2d1a10
  - White: #ffffff
- **Typography**:
  - Heading font: Playfair Display (serif)
  - Body font: Inter (sans-serif)
- **Visual Style**:
  - Luxury, professional, African hospitality-focused
  - Use subtle gold accents and maroon backgrounds
  - Clean, modern layout with generous whitespace
  - Smooth transitions and hover effects

### Home Page Content Structure
Design the home page to showcase:
1. Hero section with conference dates, location, and key value proposition
2. Trust indicators (past editions, delegates, countries, speakers)
3. What to expect (conference, exhibition, masterclasses, networking, awards)
4. Speaker highlights
5. Testimonials from past attendees
6. Ticket options
7. Final CTA to register
8. Gallery section (as requested in menu)

### Technical Requirements
- Responsive design (mobile, tablet, desktop)
- Accessible
- Fast loading
- Use the same CSS structure/variables as landing.html
